# Frontend CSVtoDB Printer Audition

Interface React + Vite + TypeScript + Tailwind CSS com componentes no padrão shadcn/ui.

## Configuração

1. Instale dependências com `npm install`.
2. Defina `VITE_API_URL` se o backend não estiver em `http://localhost:3333`.
3. Rode `npm run dev`.

## Telas

- Login
- Dashboard com filtros, métricas, gráficos e tabela paginada
- Importações por upload ou pasta configurada no backend
- Impressoras sincronizadas via WMI
